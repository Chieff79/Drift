# Drift

Multi-platform secure connection client for iOS, Android, macOS, Windows, and Linux.

Built with Flutter, powered by [sing-box](https://github.com/SagerNet/sing-box).

## Features

- One-tap secure connection
- Speed test
- Multiple server locations
- Auto server selection
- TUN mode support
- Remote profile management

## Platforms

| Platform | Status |
|----------|--------|
| iOS      | TestFlight |
| Android  | APK / AAB |
| macOS    | DMG / PKG |
| Windows  | EXE / MSIX |
| Linux    | DEB / AppImage |

## Build

```bash
flutter pub get
flutter build <platform>
```
